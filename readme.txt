== Description ==

Manage tasks for yourself or within a group, company, organization, etc.
Create projects or teams, then add tasks and assign to users.
Add attachements, discuss tasks and follow their status.

== Installation ==

1. Upload `tasks.zip` to the `/wp-content/plugins/` directory
2. Unzip tasks (should create tasks/ folder within plugins/)
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Insert shortcode `[task_all]` in any page or post (or insert `<?php do_action('task_all'); ?>` in your template)
5. Create at least one project through the 'Tasks' menu in WordPress dashboard.

Note: if your theme inserts unwanted line breaks in Task's output, then use this shortcode combination: `[raw][task_all][/raw]`
